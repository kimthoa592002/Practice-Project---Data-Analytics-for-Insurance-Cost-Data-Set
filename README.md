# README
# 🏥 Insurance Cost Analysis Project

## 📌 Project Scenario
In this project, we perform data analytics on a medical insurance charges dataset.  
This dataset is a modified version of the *Medical Insurance Price Prediction* dataset, available under the CC0 1.0 Universal License on Kaggle.

---

## 📊 Dataset Description

The dataset includes the following features:

- **Age**: Age of the insured (Integer)
- **Gender**: Encoded as:
  - Female → 1  
  - Male → 2
- **BMI**: Body Mass Index (Float)
- **No_of_Children**: Number of children (Integer)
- **Smoker**: Encoded as:
  - Smoker → 1  
  - Non-smoker → 2
- **Region**: Encoded as:
  - Northwest → 1  
  - Northeast → 2  
  - Southwest → 3  
  - Southeast → 4
- **Charges**: Medical insurance cost in USD (Float)

---

## 🎯 Objectives

The main goals of this project are:

- Load and process the dataset using pandas
- Clean the data and handle missing values
- Perform Exploratory Data Analysis (EDA)
- Identify key factors affecting insurance charges
- Build:
  - Simple Linear Regression models
  - Multiple Linear Regression models
- Improve model performance using Ridge Regression
